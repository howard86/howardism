## Summary

<Link href="/articles/anthropic">Anthropic</Link>'s agentic coding product, created by <Link href="/articles/boris-cherny">Boris Cherny</Link> in late 2024 inside an internal incubator (Anthropic Labs). Initially type-ahead-replacement; pivoted to agentic with Sonnet 3.5; pre-PMF for ~6 months until Opus 4 (May 2025) drove inflection; subsequent inflections at Opus 4.5, 4.6, and 4.7. As of May 2026 it ships across CLI, desktop, web, mobile, and IDE (VS Code, JetBrains) — same product surface, different entry points.

## Codebase

Per <Link href="/articles/boris-cherny">Boris Cherny</Link> (publicly leaked in March 2026 via human error during a release-process PR — subsequently hardened, no malicious leak), the implementation is unremarkable: TypeScript + React. The choice was deliberate: "very on-distribution for the model" at a time (late 2024) when language coverage mattered.

Boris claim: "100 lines of code a year from now" — see <Link href="/articles/harness-shrinkage-as-models-improve">Harness Shrinkage as Models Improve</Link> for the direction, not the literal claim.

## Surface forms

- **CLI** — primary surface, gets new features first. "The most powerful of all the tools" (Cat Wu).
- **Desktop app** — preview-pane integrations for front-end work; better for non-technical users.
- **IDE extensions** — VS Code, JetBrains; meaningfully smaller share of usage than CLI per Boris's audience-poll.
- **Mobile + web** — kicking off tasks AFK; Boris's daily driver.
- **Sibling: <Link href="/articles/cowork">Cowork</Link>** — same primitives, non-code outputs.

## Notable features

- `/loop` — Claude-scheduled cron job; primary primitive of <Link href="/articles/agent-loop-pattern">Agent Loop Pattern</Link>
- `/powerup` — onboarding tour through the 100+ features that exist
- Sub-agents — token-isolated context windows that report summaries; see <Link href="/articles/context-window-smart-zone">Context Window Smart Zone</Link>
- Skills — markdown files in repo that Claude can pull on demand; see push/pull in <Link href="/articles/deep-modules-for-agents">Deep Modules for Agents</Link>
- Auto mode — classifier-based permission gating; see <Link href="/articles/claude-code-auto-mode">Claude Code Auto Mode</Link>
- Routines — server-side `/loop`; survives laptop closure
- Code review — slash command + multi-agent review pattern, made reliable around Opus 4.5/4.6 per Cat Wu
