## Summary

Creator and tech lead of <Link href="/articles/claude-code">Claude Code</Link> at <Link href="/articles/anthropic">Anthropic</Link>. Engineer-by-background, author of *Programming TypeScript*. Joined Anthropic late 2024 via internal incubator "Anthropic Labs" (the same incubator that produced <Link href="/articles/claude-code">Claude Code</Link>, MCP, and the Anthropic desktop app, then disbanded — re-formed under Mike Krieger for "round two" by mid-2026). Most-listened guest in Lenny's Podcast history at time of Cat Wu's appearance (April 2026).

## Notable claims and stances

- **Coding is solved (for him).** Writes 100% of his code via <Link href="/articles/claude-code">Claude Code</Link>; has logged days of 150 PRs. Caveat: not yet true for very large codebases or off-distribution languages.
- **Phone-driven workflow.** Most work now done from the iOS Claude app; runs ~5–10 sessions concurrently, "a few hundred" agents during day, "a few thousand" overnight.
- **Loops as the future.** Heavy user of `/loop` (Claude-scheduled cron jobs) for PR babysitting, CI healing, Twitter feedback clustering. Sees the <Link href="/articles/agent-loop-pattern">Agent Loop Pattern</Link> and parallel-agent fan-out as the next harness primitive (see <Link href="/articles/harness-shrinkage-as-models-improve">Harness Shrinkage as Models Improve</Link>).
- **Build for the next model.** Built Claude Code in late 2024 knowing it had no PMF; PMF arrived with Opus 4 in May 2025 and inflected with each subsequent release (4.5 → 4.6 → 4.7).
- **Generalists across disciplines.** Every member of the <Link href="/articles/claude-code">Claude Code</Link> team writes code — including the engineering manager, PM, designers, data scientist, finance, user researcher (see <Link href="/articles/engineer-pm-convergence">Engineer PM Convergence</Link>).
- **Printing press analogy.** Sees <Link href="/articles/printing-press-software-democratization">Printing Press Software Democratization</Link> as the right historical parallel for AI-assisted software building.

Cat Wu (Head of Product for <Link href="/articles/claude-code">Claude Code</Link> and <Link href="/articles/cowork">Cowork</Link>) describes Boris as "the product visionary" setting a 3–6 month vision; she translates it into a near-term path and handles cross-functional alignment.
