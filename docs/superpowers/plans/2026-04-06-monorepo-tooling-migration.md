# Monorepo Tooling Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use otp:subagent-driven-development (recommended) or otp:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate the howardism monorepo from pnpm/Lerna/Nx/ESLint/Prettier to Bun/Turborepo/Changesets/Ultracite across three sequential PRs.

**Architecture:** Three phases executed as separate PRs: (1) Bun replaces pnpm as package manager and runtime, (2) Turborepo replaces Lerna+Nx as task runner with Changesets for publishing, (3) Ultracite replaces ESLint+Prettier for linting/formatting. Each phase is independently revertable.

**Tech Stack:** Bun, Turborepo, Changesets, Ultracite (Biome), Husky, commitlint
